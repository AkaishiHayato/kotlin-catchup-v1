## 学習のポイント

- Null 安全性を活用することで、NullPointerException を防ぐことができます
- 安全呼び出し演算子を使用することで、Null チェックを簡潔に記述できます
- レイト初期化を使用することで、非 Null 型のプロパティを柔軟に初期化できます
- Null 安全性の設計パターンを理解することで、より安全なコードを書くことができます
